📝 表达（Expression）

💬 互动（Interaction）

  • 从 "点赞 / 评论 / 分享" 到 "跟内容对话"
  • 从 "被动消费" 到 "问 AI 视频里的'为什么'"
  • AI 评论助手 → 社恐也能社交
  • 多语言桥 → 跨国互动零门槛

👥 社交（Social）

  • AI 数字分身（time-twin 思路 —— 你不在线时也能跟粉丝互动）
  • 基于内容偏好的 AI 推荐"应该认识"的创作者（不只是 follow 算法）
  • AI Memory → 关系不只是 follow，是 AI 替你管的关系图

🎨 AI 原生创作体验（AI-Native）

  • 完全 AI 生成内容（用户提示 → 视频，零拍摄）
  • AI 协同创作（用户拍一段 + AI 续写一段）
  • AI Influencer（AI 数字人创作者，无需人类）

💼 四、Mock 12 题（针对 AI 战略升级版）

A. AIGC 模型层追踪（高频，必问 2-3 题）

Q1. 你最近关注到哪个 AIGC 模型 / 趋势？为什么有意思？🔥
公式：模型 + 突破点 + 战略 implication

推荐答 OpenAI Sora 2 或 Google Veo 3（视频生成最相关 TikTok）：

"我最近一直在追 Google Veo 3。它的关键突破不是画质（Runway / Pika 也很好），而是原生支持音视频同步生成 —— 视频和配音是一次性产出的，不是后期对齐。

战略 implication 对 TikTok 意义重大：如果用户能用一句话生成带音的 15 秒视频，UGC 的门槛会被砍半。这意味着：
① 创作者生态可能裂变（更多人能创作）
② 但内容同质化风险也会上升（AI 生成模板化内容大量涌入）
③ 平台算法需要 reweight—— 区分"人创作"和"AI 生成"的权重

TikTok 现在没有自己的视频生成 base model，是接其他还是自研，这是个战略选择点。"
Q2. 你怎么追踪 AIGC 行业动态？
公式：source 多样性 + 自动化 + 输出形态
"我有自己的一套 pipeline，叫 AFI（AI Frontier Insight）：

多源采集：ArXiv（论文层）+ GitHub Trending（开源层）+ HuggingFace（模型 release 层）+ 行业 newsletter（Stratechery / Latent Space / The Information）+ Twitter（特定账号）+ 36氪 / 极客公园（中文层）

LLM 提取：DeepSeek V3 做相关性打分（10/10 = breakthrough，5/10 = noise），过滤掉 90% 噪音

分层输出：Raw Signal → Insight（这意味着什么）→ Implication（对业务什么影响），三层结构让管理层和业务负责人各取所需

所以一个新模型出来 24 小时内我能给老板一份 1 页判断。"
Q3. Sora 2 出来了，你觉得 TikTok 应该怎么 react？
公式：威胁评估 + 应对选项 + 推荐路径
  • 威胁层级：
    • 短期（6 月）：影响小（生成质量还不够 social ready）
    • 中期（1 年）：UGC 生态可能会被 reshape（生成内容大量涌入）
    • 长期（3 年）：如果 TikTok 没自己的视频 base model，可能在创作工具层失去话语权
  • 3 个应对选项：
    1. 自研（高投入）：字节有即梦在做，集成进 TikTok
    2. 合作（中等）：跟 OpenAI / Runway 合作，TikTok 提供分发，对方提供模型
    3. 专注差异化（低投入）：不竞争生成质量，专注"创作者编辑工具" + "好的素材分发算法"
  • 推荐："自研为主 + 战术合作 —— 字节有 base model 能力（豆包 / 即梦 / 海螺），整合进 TikTok 是最低战略风险路径。但同时跟头部模型（如 Veo / Sora）保持 API 兼容，给创作者更多选择。"

B. AI 应用层探索（高频，必问 2-3 题）

Q4. AI 怎么重构 TikTok 的创作 / 互动 / 社交？🔥
公式：从三个 layer 拆 + 给具体 example + 风险

创作 Layer

  • 从"录制"到"生成"：用户输入 idea → AI 生成视频草稿 → 用户微调（这是 Effect House 2.0 路径）
  • AI Co-creator：在剪辑过程中 AI 持续 suggest（"加这个转场" / "试这个配乐"）
  • 风险：AI 生成 + 模板化 → 内容同质化 → 用户审美疲劳

互动 Layer

  • AI 评论助手：帮用户用合适语气评论（社恐救星）
  • AI 直播副驾：实时给主播提示（"评论问 X，建议你回 Y"）
  • AI 多语言桥：跨国创作者跟全球粉丝实时互动

社交 Layer

  • AI 数字分身：你不在线时你的 Twin 也能跟粉丝互动（这是我自己在做的 Time Twin 项目方向）
  • AI 牵线：基于你的内容偏好，AI 推荐你"应该认识"的创作者
  • AI Memory：让你的关注关系不只是"follow"，是 AI 替你管的关系图
这一题是你的 sweet spot —— digital-twin / Time Twin / intern-clawd 都跟这条紧密相关。
Q5. AI 重构后的 TikTok 跟现在的 TikTok 最大区别是什么？
"我用三句话概括我看到的转变：

① 从"消费"到"共创" —— 现在用户主要是消费内容，未来 AI 会让更多用户成为创作者

② 从"算法分发"到"算法 + 主动表达" —— 用户跟 AI 协同表达，不只是被算法推

③ 从"短视频"到"动态体验" —— 内容会从静态发布变成可以跟你互动的 live experience（你看一个视频可以暂停问 AI 问题，可以跟视频里的虚拟人对话）

但坦诚说，这些转变都需要 base model 能力跟得上。如果模型能力还不足以让 AI 生成内容达到'social ready'，这些都是 vapor。"
